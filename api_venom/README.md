# Venom API Test Framework

Framework in the folder `api_venom` utilizing [ovh/venom](https://github.com/ovh/venom).

## Installation

You need Docker or Go installed [Go installed](https://go.dev/doc/install) to run these tests.

## Using Docker

```bash
make build
make test-ci
```

## Running locally (requires Go)
```bash
make test
```
